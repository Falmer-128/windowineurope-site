Инструкция для V2Box (iOS)

1. Установите V2Box из App Store.

2. Скопируйте ваш VLESS-ключ из нашего Telegram-бота.

3. В приложении перейдите в Config -> Routing Settings -> Add Rule.

Настройте правило для обхода РФ:

  • Outbound: direct
  • Domain: geosite:ru
  • IP: geoip:ru

Нажмите Save.
